# VanishSystem
## About:
VanishSystem is a minecraft plugin that support's 1.8.8 mc version for Paper/Spigot/Bukkit Servers Made By Herex_777.
This plugin is a Remake of Hypixel Vanish System.

## Features:
- Vanish ActionBar ✔
- Vanish command ✔
- Like Hypixel ✔
